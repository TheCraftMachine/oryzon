# Page : Contact / RDV `/contact`

## Objectif
Page de conversion finale. Simple, rassurante, sans friction. Le visiteur arrive ici prêt à agir — ne pas l'en décourager avec un formulaire complexe.

## Structure

### 1. Headline rassurant
- Titre : *"Parlons de votre projet."*
- Sous-titre : "Premier rendez-vous gratuit et sans engagement. On écoute, on conseille, vous décidez."

### 2. Calendly embed
- Widget Calendly intégré directement dans la page (pas de redirect)
- Type de RDV : "Rendez-vous découverte — 30 min"

### 3. Coordonnées directes (sidebar ou sous le widget)
- Téléphone (à remplir)
- Email (à remplir)
- Zone d'intervention : Caen et Calvados

### 4. Micro-rassurance
- "Réponse sous 24h garantie"
- "Aucun engagement lors du premier échange"
